1. Укажите след. команды:

	-для полной очистки индекса: git reset

	-для просмотра разницы между раб. каталогом и последним 
	 коммитом: git diff

	-для создания новой ветки и одновременного
	 переключения на нее: git checkout -b новая_ветка

	-для сравнения веток между собой: git diff branch1..branch2

	-для удаления локальной ветки: git branch -d название_ветки

	-для слияние веток: git merge название_ветки

	-для перезаписи последнего коммита: git commit --amend
