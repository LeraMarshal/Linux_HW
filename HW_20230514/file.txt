> Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

- зайти в нужный репозиторий на гитхабе
- нажать кнопку <>Code
- скопировать https или ssh ссылку
- в терминале выполнить команду 'git clone ссылка'

> Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

- в терминале зайти в нужную папку с помощью команды 'cd'
- выполнить 'git init' для инициализации гит-репозитория
- выполнить 'git remote add origin <URL репозитория>', чтобы связать локальный репозиторий с удаленным репозиторием на GitHub
- выполнить 'git push -u origin master', для загрузки локального репозитория на GitHub

> Как получить на локальный компьютер изменения с github.com

- в терминале зайти в нужную папку с помощью команды 'cd'
- выполнить 'git pull', для загрузки последних изменений из удаленного репозитория

> Какую информацию нам дают команды:

git status
- показывает текущий статус локального Git-репозитория
    (какие файлы были изменены, добавлены или удалены, какие изменения еще не были зафиксированы, и какие изменения готовы к зафиксированию,
    а также сообщает о текущей ветке, на которой находимся)

git log
- показывает историю коммитов локального репозитория

git remote -v
- отображает список удаленных репозиториев, связанных с локальным Git-репозиторием, и их URL-адреса
